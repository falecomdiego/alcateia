# Guia de uso - Mapa da Noite

## O que e esta versao

Esta e a versao publica do painel Mapa da Noite. Ela foi preparada para apresentacao externa, avaliacao academica e leitura por pessoas que precisam compreender o valor do projeto sem acessar bastidores tecnicos.

## Como abrir

Abra o arquivo `index.html` em um navegador.

Para compartilhamento, mantenha a pasta inteira junto com:

- `index.html`
- `styles.css`
- `app.js`
- `data/dados_publicos_dashboard.json`
- `data/dados_publicos_dashboard.js`

## Como ler o painel

1. Comece pela secao Inicio para entender a narrativa geral.
2. Leia Base real de escuta para ver a dimensao da rodada.
3. Use O que os comentarios revelam para observar sinais positivos, sinais criticos e temas recorrentes.
4. Compare Operacao x Experiencia para separar atritos operacionais de percepcao do publico.
5. Leia Evidencias representativas para ver exemplos anonimizados de comentarios reais.
6. Consulte Recomendacoes acionaveis para conectar problema, evidencia, recomendacao, impacto esperado e indicador.
7. Feche com Limites da rodada para evitar interpretacoes absolutas.

## Cuidados de interpretacao

- Os resultados indicam padroes observados na amostra.
- A rodada inicial nao deve ser lida como conclusao definitiva sobre toda a cena noturna.
- Os recortes observados nao devem ser interpretados como ranking absoluto de melhores ou piores festas.
- Evidencias foram anonimizadas e curadas para apresentacao publica.
- Bastidores tecnicos, IDs, arquivos, links, logs e caminhos locais foram removidos da interface publica.
