# Pacote de entrega final - rodada inicial

## Finalidade

Este guia orienta a ordem de consulta dos arquivos finais aprovados da rodada inicial. A entrega considera apenas lote_01, lote_02 e lote_03.

## Ordem recomendada de consulta

1. `analise_comentarios_evento\12_leitura_executiva_final\leitura_executiva_final_rodada_inicial.docx`

   Documento principal para leitura executiva. Deve ser consultado primeiro, pois organiza o escopo, o metodo de transformacao dos comentarios em evidencias, os macroproblemas, impactos, prioridades, responsaveis sugeridos, indicadores, limites da amostra e proximos passos.

2. `analise_comentarios_evento\11_painel_final_rodada_inicial\painel_final_rodada_inicial.xlsx`

   Arquivo de consulta visual e operacional do painel final da rodada inicial. Deve ser usado para navegar pelos resultados consolidados apos a leitura executiva.

3. `analise_comentarios_evento\11_painel_final_rodada_inicial\matriz_executiva_agrupada_rodada_inicial.xlsx`

   Base executiva agrupada validada. Deve ser usada para conferir a origem dos macroproblemas, volumes, prioridades, eixos, natureza dos problemas, evidencias e indicadores.

4. `analise_comentarios_evento\12_leitura_executiva_final\nota_metodologica_rodada_inicial.docx`

   Documento de apoio metodologico. Deve ser usado para explicar origem dos dados, divisao por lotes, etapas do pipeline, criterios de rastreabilidade, limites da amostra e confirmacao de que lote_04 e lote_05 nao fazem parte deste ciclo.

5. `analise_comentarios_evento\12_leitura_executiva_final\briefing_apresentacao_rodada_inicial.docx`

   Estrutura de apoio para apresentacao. Deve ser consultado depois da leitura executiva e da nota metodologica, para organizar a comunicacao dos resultados.

6. `analise_comentarios_evento\10_consolidacao_rodada_inicial\matriz_resolutiva_rodada_inicial.xlsx`

   Matriz resolutiva consolidada. Deve ser usada como arquivo de auditoria resolutiva, com problemas, evidencias, recomendacoes, responsaveis sugeridos e indicadores.

- `analise_comentarios_evento\10_consolidacao_rodada_inicial\status_pipeline_rodada_inicial.md`

## Restricoes de uso

- Nao usar arquivos de amostra como base oficial.
- Nao usar arquivos com sufixo baseada_em_amostra como base oficial.
- Nao apresentar lote_04 ou lote_05 como parte desta rodada.
- Nao apresentar a rodada inicial como leitura definitiva de 10.000 comentarios.
- Nao alterar os arquivos aprovados ao consultar o pacote.
