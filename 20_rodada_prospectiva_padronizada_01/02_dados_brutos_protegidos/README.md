# Dados brutos protegidos — MDN-RPP01

Esta pasta contém apenas orientação versionável. Os dados reais deverão ser colocados em `dados/`, caminho ignorado pelo Git.

## Regras

- aceitar somente arquivos vinculados a `fonte_id` e `coleta_id` registrados;
- nunca renomear ou alterar um XLSX depois do cálculo do hash;
- não abrir e salvar o XLSX em aplicativo que possa reescrevê-lo;
- manter inventário SHA-256;
- restringir acesso;
- manter cópia protegida em ambiente distinto;
- não copiar nomes, perfis, URLs, identificadores ou comentários para documentos públicos.

Nenhum arquivo da rodada inicial deve ser copiado para esta pasta como se fosse dado novo.
