# Publicação sanitizada — MDN-RPP01

## Status

Bloqueada até G4.

Somente arquivos gerados pelo sanitizador V2 e aprovados no registro de liberação poderão entrar em `saidas/`. A existência técnica de um arquivo não altera o status do gate.

## Conteúdo proibido

- nomes, perfis, URLs, IDs ou pseudônimos internos;
- comentários e fragmentos literais;
- rankings individualizantes;
- acusações atribuíveis;
- células com menos de cinco ocorrências;
- combinações raras que permitam busca reversa;
- causalidade, culpa, representatividade ou validação científica não demonstradas.
